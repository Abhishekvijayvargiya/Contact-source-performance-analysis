**Lead Source Performance Analysis**
This project analyzes lead data collected over a three-month period to measure marketing and sales performance.

**Features**
Lead Source Analysis
Total Leads by Source
Enrollments by Source
Conversion Rate Analysis
KPI Dashboard
Business Insights
Pivot Tables & Charts
**Tools Used**
Microsoft Excel
Pivot Tables
Pivot Charts
Data Cleaning
Business Analytics
**Key Insights**
Identified the highest lead-generating channels.
Compared enrollments across different marketing sources.
Calculated conversion rates to evaluate campaign effectiveness.
Created an interactive dashboard for quick business reporting.
Repository Topics (GitHub Tags)

This is a strong portfolio project for roles such as Data Analyst, Business Analyst, MIS Executive, CRM Analyst, Product Executive, and Sales Operations Analyst. It demonstrates practical business reporting and KPI analysis using Excel.
